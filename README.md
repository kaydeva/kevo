Kevo — AI‑Powered Career Intelligence Platform
Kevo is an intelligent career‑analysis engine built with React + Vite, designed to help users understand their strengths, personality traits, and ideal career paths through structured AI reasoning.
The platform transforms raw user inputs into a complete professional profile and delivers personalized, data‑driven career recommendations.

🚀 Features
-- AI‑Driven Personality Analysis
Kevo interprets user responses using structured reasoning models to generate accurate psychological and cognitive profiles.

-- Career Matching Engine
The system compares user traits with thousands of modern career paths and recommends the most suitable roles.

-- Fast & Modern Frontend
Built with React + Vite for instant loading, smooth animations, and a premium user experience.

-- Secure Enterprise‑Grade Backend
A custom API handles form submissions, email delivery, and data validation with strict security practices.

-- Enterprise Contact System
Companies can request access through a secure CTA form that sends messages directly to the admin inbox.

-- Tech Stack:
React + Vite — frontend framework & dev environment

Framer Motion — cinematic animations

TailwindCSS — modern styling

Node.js + Express — backend API

Nodemailer — secure email delivery

Environment Variables (.env) — protected credentials

Rate Limiting + Input Validation — backend security
